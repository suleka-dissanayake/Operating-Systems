This repository contains **practical implementations, laboratory exercises, assignments, and operating system concepts** developed during my **2nd Year, 2nd Semester** as part of the **Bachelor of Science in Information Technology (BSc. in IT)** program at the **University of Vavuniya, Sri Lanka**.

It focuses on the **Operating Systems** module, covering fundamental concepts such as **process management, CPU scheduling, synchronization, memory management, deadlocks, file systems, and Linux system programming**, while strengthening problem-solving and systems programming skills using **C**.
